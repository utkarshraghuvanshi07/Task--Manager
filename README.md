# ✅ Task Manager JS

A simple Task Manager built using **HTML**, **CSS**, and **Vanilla JavaScript**.

This project was created to practice JavaScript fundamentals by building a dynamic task management interface without using any frameworks or libraries.

> 🚧 **Status:** Under Development

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit tasks inline
- 🗑️ Delete tasks
- ⚡ Dynamic task rendering
- 📝 Input validation (prevents empty tasks)
- 🎯 Event Delegation for efficient event handling

---

This project helped me practice:

## 🚀 Getting Started

1. Clone this repository

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or external libraries are required.
